### Características
- Integración de servicios con retrofit
- Integración de corrutinas
- Desarrollada con Arquitectura MVVM

### Uso
Para poder instalar la aplicación, se encuentra un .apk disponible en:
```sh
apk/tvmaze-1.0.0.apk
```
